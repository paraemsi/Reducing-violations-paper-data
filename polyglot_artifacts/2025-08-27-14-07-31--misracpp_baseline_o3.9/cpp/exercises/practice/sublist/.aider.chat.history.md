#### # Instructions  
####   
#### Given any two lists `A` and `B`, determine if:  
####   
#### - List `A` is equal to list `B`; or  
#### - List `A` contains list `B` (`A` is a superlist of `B`); or  
#### - List `A` is contained by list `B` (`A` is a sublist of `B`); or  
#### - None of the above is true, thus lists `A` and `B` are unequal  
####   
#### Specifically, list `A` is equal to list `B` if both lists have the same values in the same order.  
#### List `A` is a superlist of `B` if `A` contains a sub-sequence of values equal to `B`.  
#### List `A` is a sublist of `B` if `B` contains a sub-sequence of values equal to `A`.  
####   
#### Examples:  
####   
#### - If `A = []` and `B = []` (both lists are empty), then `A` and `B` are equal  
#### - If `A = [1, 2, 3]` and `B = []`, then `A` is a superlist of `B`  
#### - If `A = []` and `B = [1, 2, 3]`, then `A` is a sublist of `B`  
#### - If `A = [1, 2, 3]` and `B = [1, 2, 3, 4, 5]`, then `A` is a sublist of `B`  
#### - If `A = [3, 4, 5]` and `B = [1, 2, 3, 4, 5]`, then `A` is a sublist of `B`  
#### - If `A = [3, 4]` and `B = [1, 2, 3, 4, 5]`, then `A` is a sublist of `B`  
#### - If `A = [1, 2, 3]` and `B = [1, 2, 3]`, then `A` and `B` are equal  
#### - If `A = [1, 2, 3, 4, 5]` and `B = [2, 3, 4]`, then `A` is a superlist of `B`  
#### - If `A = [1, 2, 4]` and `B = [1, 2, 3, 4, 5]`, then `A` and `B` are unequal

#include "sublist.h"

#include <algorithm>  // for std::search
#include <vector>

namespace sublist {

static bool contains(const std::vector<int>& haystack,
                     const std::vector<int>& needle) {
    // An empty sequence is always found.
    if (needle.empty()) {
        return true;
    }
    // std::search returns end iterator if not found.
    return std::search(haystack.begin(), haystack.end(),
                       needle.begin(), needle.end()) != haystack.end();
}

result sublist(const std::vector<int>& list_a, const std::vector<int>& list_b) {
    if (list_a == list_b) {
        return result::equal;
    }

    if (list_a.size() < list_b.size()) {
        return contains(list_b, list_a) ? result::sublist : result::unequal;
    }

    if (list_a.size() > list_b.size()) {
        return contains(list_a, list_b) ? result::superlist : result::unequal;
    }

    // Same size but not equal.
    return result::unequal;
}

}  // namespace sublist
